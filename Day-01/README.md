# 📖 Overview

This is the **Day 01** program in my **JavaScript Series**.

The program checks whether numbers from **0 to 9** are **even** or **odd** using a **for loop** and an **if...else** statement.

This exercise helps beginners understand the fundamentals of JavaScript programming and logical decision-making.

---

# 📚 Topics Covered

- ✅ Variables (`let`)
- ✅ `for` Loop
- ✅ `if...else` Statement
- ✅ Modulus Operator (`%`)
- ✅ `console.log()`
- ✅ Program Execution using Node.js

---

# 💻 Program

```javascript
let count = 10;

for (let i = 0; i < count; i++) {
    if (i % 2 === 0) {
        console.log(i + " is even");
    } else {
        console.log(i + " is odd");
    }
}
```
