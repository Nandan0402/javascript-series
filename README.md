# 🚀 Project Overview

The **JavaScript Series** is a structured learning repository designed to help beginners master JavaScript through daily practice.

This repository covers JavaScript fundamentals, ES6 concepts, DOM manipulation, events, asynchronous programming, APIs, and hands-on mini projects.

The goal is to build a strong programming foundation while creating practical applications.

### Suitable For

- 🎓 Students
- 💡 JavaScript Beginners
- 💼 Portfolio Building
- 🌐 Frontend Development
- 🚀 Web Development Interviews

# ▶️ How to Run

### Clone Repository

```bash
git clone https://github.com/Nandan0402/javascript-series.git
```

### Open Project

```bash
cd javascript-series
```

### Run with Node.js

```bash
node filename.js
```

Or simply open the HTML file in your browser for DOM-based projects.
